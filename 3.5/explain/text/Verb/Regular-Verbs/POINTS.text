	(REGULAR VERB)      

	POINTS sets the plotting mode for the SHOW and PLOT display to
	points mode.

	No adverbs are used by POINTS.

	ADVERB SET:
	
	STATUS(3)    Type of SHOW or PLOT plot as determined by the
		     last LINE, HISTOGRAM, or POINTS command. 
 
		     Values:  0 = line; 1 = histogram; 2 = points


	EXAMPLE:

	To display the current scan in points mode, specify;

			POINTS PAGE SHOW
