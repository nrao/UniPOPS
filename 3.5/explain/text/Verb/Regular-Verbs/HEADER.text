	(REGULAR VERB)

	HEADER is an informational verb which prints some of the header
	information for the scan in Array (0).  The format of the output
	is telescope dependent.

	No adverbs are used or set by HEADER.

	EXAMPLE:

	To print the header information on the text screen, specify:

		GET 3245 
		HEADER

	and self explanatory information about the scan will appear on
	the text screen.
