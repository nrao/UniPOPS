	(REGULAR VERB)

	Sets plotting mode for RESHOW displays to POINTS.

	No adverbs are used by RPOINTS.

	ADVERB SET:
	 
	STATUS(4) :  Type of RESHOW plots as determined by the last
		     RLINE, RHIST, or RPOINTS command.  

		     Values: 0 = line; 1 = histogram; 2 = points

	EXAMPLE:

	Your last display was drawn using LINE, and you want to see the
	effect of recent changes using RESHOW and a different plotting
	mode, so you specify:

		RPOINTS  RESHOW
